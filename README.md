
# **Spanish Keyboard Hotkeys**

*(SHIFT / CAPSLOCK will alter these to capitals)*

### Default Windows Hotkeys:
    AltGr + a = á
    AltGr + e = é
    AltGr + í = i
    AltGr + o = ó
    AltGr + u = ú
       
### Additional Hotkeys in AHK Script:
    AltGr + y = ü
    AltGr + n = ñ
    AltGr + 1 = ¡ 
    AltGr + / = ¿  (AKA. the ? key)
